## Instalation
Use the package manager [pip](https://pip.pypa.io/en/stable/) to install requirements. 
<br>
For example: 
<br>

```bash
pip install dash
```

<br>

```bash

```

<br>

```bash
source yourfoldername/bin/activate
```

Clone the repository and well done, that's all.
